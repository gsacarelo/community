How to Use This Weather Page

📍 Location
• Use "Search" to enter a city name.
• Use "GPS" to detect your current location automatically.

⏱ Time Range
• 24h / 48h / 72h / 96h adjusts how far into the future the chart shows.

📊 Graph
• Solid line = Current temperature forecast.
• Dashed line = Yesterday’s temperature (shifted for comparison).
• First vertical line = Current time.
• Extra vertical lines = Midnight markers.

🧠 Smart Cards
• Hover over a card to highlight its line on the chart.
• Click a card to lock the highlight.

💾 Save
• Save downloads the chart as a PNG image.
